# Guardian — Research Library for the Marblehead Independent

A curated collection of research reports, case studies, academic papers, and
practitioner presentations on sustaining independent journalism. This material
supports strategy and operations for the **Marblehead Independent**, with a
particular focus on reader revenue and the Guardian's contribution/membership
model.

## Repository structure

```
research/
├── guardian-model/          How the Guardian built its reader-funded model
├── reader-revenue/          Subscriptions, paywalls, pricing, and conversion
├── local-news/              Local and community journalism case studies
├── trust-and-transparency/  Building reader trust (and why it pays)
└── fundraising/             Nonprofit fundraising benchmarks
```

## Contributing

When adding a document:

1. Place it in the most relevant `research/` subfolder (create a new one if no
   category fits).
2. Rename the file to a lowercase, hyphenated, descriptive name that includes
   the author/publisher and year where known.
3. Add a row to the matching index table above with a one-line description.
